ManyChat / WhatsApp

● Ativo

Quando o cliente envia "consulta" no WhatsApp, o ManyChat coleta nome, data e horário e envia via webhook para este sistema.

POST /api/webhook/consulta
{
 "nome": "{{nome_cliente}}",
 "telefone": "{{telefone}}",
 "data": "{{data}}",
 "hora": "{{horario}}",
 "origem": "whatsapp"
}
⚙️

Make (Integromat)

⚠ Configurar

Crie um cenário no Make com o gatilho WhatsApp Business → filtrar → HTTP POST para o webhook deste sistema.

1. Trigger: WhatsApp New Message
2. Filter: contains "consulta"
3. HTTP POST → /api/webhook
4. Google Calendar: Create Event
🤖

Fluxo de Resposta Automática

Configure o ManyChat para confirmar o recebimento e avisar que o Dr. Edmar entrará em contato para confirmar.

Gatilho: "consulta" / "agendar"
→ Coleta: Nome
→ Coleta: Data (dd/mm)
→ Coleta: Horário
→ Webhook → Sistema
→ Resposta: "Recebido! ✅
  Dr. Edmar confirma em breve."
📲

Registro Presencial

Para atendimentos no escritório, o estagiário registra diretamente. A consulta é sincronizada automaticamente com o Google Calendar.

Consultas aparecem com a badge Escritório para identificar a origem.

✅ Conectado: dr.edmarsilva@gmail.com

Sincronização Automática

Toda consulta confirmada é adicionada automaticamente ao Google Calendar do Dr. Edmar com dados completos do cliente.

Evento: "Consulta — [Nome]"
Descrição: Tipo + Origem + Tel
Duração: 1h (configurável)
Notificação: 30min antes
Calendário: Principal
